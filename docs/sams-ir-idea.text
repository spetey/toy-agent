
So Sam Eisenstat says we can get reversibility by tracking a special "instruction register" that basically buffers the next instruction value for a step.

## Problem case: `tape[tape[CL]] == 11`

When the data in `CL` happens to point to an address with a `J`, then it's ambigous whether the current IP is from a jump or not.

Call this state `A` (for "ambiguous"):

```
Adx:    0    1    2    3    4    5    6    7
Val:    0   11    0    1    0    1    2    0
Ins:    0    J    0    <    0    <    >    0
                           IP   CL
     H0,H1
```

`A` could have been preceded by

```
Adx:    0    1    2    3    4    5    6    7
Val:    0   11    0    1    0    1    2    0
Ins:    0    J    0    <    0    <    >    0
                      IP             CL
     H0,H1
```

and then just incremented after performing `<`, or by 

```
Adx:    0    1    2    3    4    5    6    7
Val:    0   11    0    1    0    4    2    0
Ins:    0    J    0    <    0    }    >    0
            IP                  CL
     H0,H1
```

and then jumping.

Would an IR help with this?

Let's trace the history where `A` was preceded by `J` not `<`, with an `IR`.
